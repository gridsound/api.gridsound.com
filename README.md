# api.gridsound.com

A Docker Compose file creating a development environment with MySQL, phpMyAdmin and a PHP Web server.

## Prerequisite

- [Docker Compose](https://docs.docker.com/compose/install/)

## Usage

OS X & Linux & Windows:

```sh
docker-compose up
```

## Usage example

Access to `phpMyAdmin` on: 
```
http://localhost:8001
```

URL API:
```
localhost:8000/<route>.php
```
